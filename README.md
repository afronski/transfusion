# transfusion #

![Logo](https://raw.github.com/afronski/transfusion/master/Doc/img/Icon.png)

Really small graphics engine written in 2009. Not maintained.

Technologies and languages used in project:

- C++
- Cg
- FreeImage
- OpenGL, glew, glut
